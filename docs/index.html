Condition variable

In JavaScript cooperative multi-coroutine scheduling, a mutex is optional because event loop cannot be switched between the condition checking and the wait.

import { ConditionVariable, Mutex } from 'coroutine-locks';
const cv = new ConditionVariable();
const mutex = new Mutex();

await mutex.lock();
while (!condition) await cv.wait(mutex);
mutex.unlock();

// mutex is optional
while (!condition) await cv.wait();

cv.signal();
cv.broadcast();
